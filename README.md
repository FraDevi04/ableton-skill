# Making Music Skill

Knowledge base from **Making Music: 74 Creative Strategies for Electronic Music Producers** by Dennis DeSantis.

## Structure
- SKILL.md - Master index with Core Frameworks, Chapter Index, Topic Index
- chapters/ - 74 chapter summaries (ch01-ch74)
- glossary.md - 95 key terms with definitions
- patterns.md - All techniques organized by category
- cheatsheet.md - Quick reference tables

## How to Use
- Load without arguments for core frameworks reference
- Ask about a topic (e.g. "repetition", "arranging")
- Ask for a specific chapter (e.g. "ch01")

## Book Sections
- **Problems of Beginning** (ch01-ch25): Starting, active listening, constraints
- **Problems of Progressing** (ch26-ch61): Harmony, melody, rhythm, arrangement
- **Problems of Finishing** (ch62-ch74): Dramatic arc, endings, feedback, finishing

## Author
Dennis DeSantis - electronic music producer, composer, and educator
