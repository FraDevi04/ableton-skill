# Chapter 58: Maximal Density

## Core Idea
Start with the densest possible section (everything playing). Then subtract elements to create variations. Each density level becomes a song section. Works especially well in minimal techno where sections are similar.

## Frameworks Introduced
- **Maximal density**: Start with EVERYTHING playing at once — the densest possible section. This is your "peak" section.
- **Subtractive variation**: Subtract elements from the maximal section to create variations. Each density level becomes a song section.
- **Minimal techno application**: This works especially well in minimal techno where sections are similar but differ in density.

## Key Concepts
- **Peak first**: Start with the peak (maximal density) and work backwards
- **Density as section**: Different density levels = different sections
- **Subtraction over addition**: It's easier to remove elements than to add them meaningfully
- **Similar sections**: In minimal genres, sections are similar — density is the main differentiator

## Mental Models
- Use maximal density when you're struggling to create section contrast
- Think of density as your primary arrangement tool
- Start at the peak and work backwards

## Anti-patterns
- **Adding instead of subtracting**: Adding elements to a sparse section is harder than subtracting from a dense one
- **Too many density levels**: Creating too many similar sections creates monotony
- **Ignoring the peak**: The maximal density section should be the most exciting

## Key Takeaways
1. Start with the densest possible section — everything playing
2. Subtract elements to create variations — each density level becomes a section
3. Works especially well in minimal techno where sections are similar
4. Subtraction is easier and more effective than addition for section creation
5. The maximal density section should be the peak of the song
