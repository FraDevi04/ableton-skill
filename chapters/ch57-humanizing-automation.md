# Chapter 57: Humanizing With Automation Envelopes

## Core Idea
Automate attack/decay for subtle duration changes. Tiny tuning variations (cents) simulate live intonation. Apply to any parameter, not just filter. Randomization can achieve similar effects.

## Frameworks Introduced
- **Envelope automation**: Automate attack and decay parameters to create subtle duration changes. This simulates the natural variation of live performance.
- **Tuning variation**: Apply tiny tuning variations (cents) to simulate live intonation. No live performer plays perfectly in tune — artificial perfection sounds sterile.
- **Parameter universality**: Apply automation to ANY parameter, not just filter. Volume, pan, reverb, delay — any parameter can be automated for humanization.
- **Randomization**: Randomization can achieve similar humanizing effects — but you must evaluate every result.

## Key Concepts
- **Duration variation**: Natural performers vary note durations subtly — automate attack/decay to simulate this
- **Intonation variation**: Live performers vary pitch subtly — automate tuning to simulate this
- **Any parameter**: Humanization isn't limited to filter — apply to any parameter
- **Random vs. deliberate**: Randomization is fast but requires evaluation; deliberate automation is slower but more controlled

## Mental Models
- Use automation humanization when your music sounds too rigid or artificial
- Think of every parameter as potentially humanizable
- Randomization is a starting point — you must evaluate and curate

## Anti-patterns
- **Automating everything**: Not every parameter needs humanization
- **Over-humanizing**: Too much variation creates instability
- **Ignoring random results**: Randomization produces bad results — evaluate everything

## Key Takeaways
1. Automate attack/decay for subtle duration changes
2. Apply tiny tuning variations (cents) to simulate live intonation
3. Apply to ANY parameter, not just filter
4. Randomization can achieve similar effects — but evaluate every result
5. Humanization makes electronic music feel more alive and less artificial
