## Key Concepts
- **Scale as foundation**: All harmony is built on scales
- **Diatonic harmony**: Chords built from the notes of a single scale
- **Function**: Each chord has a role (tonic, dominant, pre-dominant)
- **Substitution**: Chords can substitute for each other based on functional similarity
