# Chapter 48: Sound-Color Melody

## Core Idea
Distribute a melody between multiple instruments. Split by pitch range (high voice / low voice). Use instruments with mirrored envelopes (fast attack/gradual decay + slow attack). Smooth, seamless timbral exchange.

## Frameworks Introduced
- **Klangfarbenmelodie** (Sound-Color Melody): Distribute a single melody between multiple instruments. The melody changes timbre as it progresses — the listener perceives a single melody with shifting color.
- **Pitch range splitting**: Split the melody by pitch range — high voice on one instrument, low voice on another.
- **Mirrored envelopes**: Use instruments with complementary envelopes — one with fast attack/gradual decay, another with slow attack/fast decay. The transition between instruments is seamless.

## Key Concepts
- **Timbral exchange**: The melody passes from one instrument to another — the pitch may be continuous but the timbre changes
- **Seamless transition**: Mirrored envelopes create smooth transitions between instruments
- **Pitch range vs. time**: Split by pitch range (high/low) or by time (instrument A plays first half, instrument B plays second half)
- **Perceptual unity**: The listener perceives a single melody despite timbral changes

## Mental Models
- Use sound-color melody when you want a melody that feels alive and shifting
- Think of timbre as a melodic parameter — the melody has pitch AND color
- Mirrored envelopes create seamless timbral transitions

## Anti-patterns
- **Abrupt timbral changes**: Without mirrored envelopes, timbral transitions are jarring
- **Too many instruments**: Distributing between too many instruments creates confusion
- **Ignoring the envelope**: The envelope shape determines whether the transition is smooth

## Key Takeaways
1. Distribute a melody between multiple instruments — the melody changes timbre as it progresses
2. Split by pitch range (high voice / low voice) or by time
3. Use instruments with mirrored envelopes for seamless transitions
4. The listener perceives a single melody despite timbral changes
5. Timbre is a melodic parameter — the melody has pitch AND color
