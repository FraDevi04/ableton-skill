# Chapter 16: Procrastination and Timeboxing

## Core Idea
Set fixed short timeboxes (20 minutes) for intimidating tasks. Stop when the timer goes off, even if in flow. This breaks large tasks into manageable chunks and builds momentum.

## Frameworks Introduced
- **Timeboxing**: Set a fixed time limit (e.g., 20 minutes) for any task. When the timer goes off, STOP — even if you're in the middle of something.
- **Pomodoro-style**: The technique is similar to the Pomodoro method — short focused bursts with mandatory breaks.
- **Momentum building**: Starting is the hardest part. Once you've started, continuing is easier. The timebox makes starting less intimidating.

## Key Concepts
- **Intimidation of large tasks**: "Write a song" is overwhelming; "work for 20 minutes" is not
- **The stop is the point**: Stopping at the timer forces you to return later, creating a habit loop
- **Flow states**: Sometimes you'll be in so much flow you don't want to stop — the timer discipline still applies
- **Task decomposition**: Any large task can be broken into 20-minute chunks

## Mental Models
- Use timeboxing when facing an intimidating creative task
- Think of it as "just start" — the 20-minute limit removes pressure
- The mandatory stop creates a return incentive — you'll want to continue

## Anti-patterns
- **Ignoring the timer**: If you keep going past the timebox, you defeat the purpose
- **Making the timebox too long**: 20 minutes is the sweet spot — too long loses the intimidation-busting effect
- **Using timeboxing for everything**: Not every task needs timeboxing — use it for the ones that feel overwhelming

## Key Takeaways
1. Set a 20-minute timer for any intimidating creative task
2. STOP when the timer goes off — even if in flow
3. The stop creates a return incentive — you'll want to continue
4. Breaking large tasks into 20-minute chunks makes them manageable
5. Timeboxing is a procrastination-busting tool, not a time-management system
