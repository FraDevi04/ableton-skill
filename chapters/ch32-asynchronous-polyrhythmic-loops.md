# Chapter 32: Asynchronous or Polyrhythmic Loops

## Core Idea
Trigger one instrument with multiple clips of different lengths. They create polyrhythmic composites. Add automation loops and LFOs at different rates. Classic "Berlin School" and Brian Eno technique.

## Frameworks Introduced
- **Polyrhythmic composites**: Trigger one instrument with multiple clips of different lengths. Each clip loops at its own rate, creating complex polyrhythmic patterns.
- **Berlin School technique**: Named after the Berlin School of electronic music (Tangerine Dream, Klaus Schulze) — long, interlocking loops creating evolving patterns.
- **Brian Eno application**: Eno uses this technique extensively — multiple loop lengths creating ever-shifting patterns.
- **Additional layers**: Add automation loops and LFOs at different rates for even more complexity.

## Key Concepts
- **One instrument, multiple triggers**: A single instrument can be triggered by multiple clips
- **Loop length as polyrhythm**: Different loop lengths create mathematical polyrhythms
- **Evolution over time**: The pattern never repeats exactly (until all loops complete their LCM cycle)
- **Automation as rhythm**: Automation loops at different rates add another layer of polyrhythm

## Mental Models
- Use polyrhythmic loops when you need evolving, non-repetitive patterns
- Think of it as "clocks at different speeds" — each keeps its own time
- The longer the loop lengths, the longer before the pattern repeats

## Anti-patterns
- **Too many loops**: More loops = more complexity, but also more confusion
- **No musical purpose**: Polyrhythms should serve the music, not just demonstrate complexity
- **Ignoring the LCM**: The pattern won't repeat until all loops complete their least common multiple cycle — plan accordingly

## Key Takeaways
1. Trigger one instrument with multiple clips of different lengths
2. Each loop creates a polyrhythmic composite — the pattern evolves over time
3. Add automation loops and LFOs at different rates for more complexity
4. This is a classic Berlin School / Brian Eno technique
5. Plan for the LCM cycle — the pattern won't repeat until all loops complete
