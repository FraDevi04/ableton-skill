# Chapter 26: Foreground, Middle Ground, Background

## Core Idea
Divide musical elements into three depth layers. Foreground: high, loud, fast, rapidly changing. Middle ground: moderate activity. Background: steady, consistent. Vary depth temporarily to draw attention.

## Frameworks Introduced
- **Three-layer depth model**: Divide elements into Foreground (high, loud, fast, rapidly changing), Middle Ground (moderate), Background (steady, consistent).
- **Temporary depth shifts**: Draw attention by temporarily moving elements between layers — a background element briefly becomes foreground.
- **Depth as attention management**: The depth layer determines where the listener's attention goes.

## Key Concepts
- **Depth ≠ volume**: Depth is a combination of pitch, volume, speed, and rate of change
- **Foreground dominance**: Only one thing should be in the foreground at a time
- **Background stability**: Background provides the foundation that allows foreground to shine
- **Attention as a resource**: Listener attention is limited — depth layering manages it

## Mental Models
- Use the three-layer model when arranging or mixing
- Think of depth as an attention-management tool
- Shift elements between layers to create interest and variety

## Anti-patterns
- **Multiple foregrounds**: Having too many competing foreground elements creates confusion
- **Flat mixing**: Putting everything at the same depth level
- **Ignoring the background**: Background elements still need to be good

## Key Takeaways
1. Divide elements into three depth layers: foreground, middle ground, background
2. Foreground = high, loud, fast, rapidly changing
3. Background = steady, consistent, low-profile
4. Vary depth temporarily to draw attention — move elements between layers
5. Only one thing should be in the foreground at a time
