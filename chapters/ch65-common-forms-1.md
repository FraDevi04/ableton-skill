# Chapter 65: Common Forms 1: Elements of Song Form

## Core Idea
Verse (main body, 16-32 bars), Chorus (contrast, hook, resolution), Bridge (contrast, different key/chords, once only). Common form: ABABCB (Verse-Chorus-Verse-Chorus-Bridge-Chorus).

## Frameworks Introduced
- **Verse**: The main body of the song. Typically 16-32 bars. Contains the main lyrical content. Repetitive structure with varying lyrics.
- **Chorus**: The contrast to the verse. Contains the hook (most memorable part). Provides resolution and emotional release. Typically shorter than the verse.
- **Bridge**: A contrast section. Different key or chords from verse/chorus. Appears only once. Provides a departure from the main material.
- **Common form**: ABABCB — Verse-Chorus-Verse-Chorus-Bridge-Chorus. The most common song form in popular music.

## Key Concepts
- **Section roles**: Each section has a specific role in the song's narrative
- **Repetition vs. contrast**: Verse = repetition (same music, different lyrics). Chorus = repetition (same music, same lyrics). Bridge = contrast (different music).
- **Hook placement**: The hook is typically in the chorus — the most memorable part
- **Bridge limitation**: The bridge appears only once — using it more than once dilutes its impact

## Mental Models
- Use common song forms when you need a proven structure
- Think of each section as having a specific narrative role
- ABABCB is a reliable default — start here and vary as needed

## Anti-patterns
- **Too many bridges**: Using the bridge more than once dilutes its impact
- **No chorus**: Songs without a clear chorus can work, but they need a different structural logic
- **Ignoring the hook**: The hook is the most important part — don't bury it

## Key Takeaways
1. Verse = main body (16-32 bars), Chorus = contrast/hook/resolution, Bridge = contrast (once only)
2. Common form: ABABCB (Verse-Chorus-Verse-Chorus-Bridge-Chorus)
3. Verse provides repetition; chorus provides resolution; bridge provides contrast
4. The hook is typically in the chorus
5. ABABCB is a reliable default — start here and vary as needed
