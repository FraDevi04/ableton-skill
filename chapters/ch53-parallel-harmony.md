# Chapter 53: Parallel Harmony

## Core Idea
Each note moves the same number of semitones in the same direction. Essentially transposed copies of one chord. Common in house/deep house. Sampled chords, chord memory synths, chord-generating MIDI tools all create parallel harmony.

## Frameworks Introduced
- **Parallel harmony**: Each note of a chord moves the same number of semitones in the same direction. It's essentially a transposed copy of one chord shape.
- **House music connection**: Parallel harmony is extremely common in house and deep house music — the "chord memory" synth effect is parallel harmony.
- **Tool-generated parallel harmony**: Sampled chords, chord memory synths, and chord-generating MIDI tools all naturally produce parallel harmony.

## Key Concepts
- **Transposed copy**: Parallel harmony is a chord shape moved up or down by the same interval for each note
- **Voice leading violation**: Parallel harmony violates traditional voice leading (parallel 5ths and 8ves are forbidden in classical)
- **Genre-specific**: Parallel harmony works well in electronic genres but may sound "wrong" in classical contexts
- **Sampled chords**: When you sample a chord and play it at different pitches, you get parallel harmony

## Mental Models
- Use parallel harmony when you want a simple, powerful chord progression
- Think of it as "moving a chord shape" up and down the keyboard
- It's genre-specific — great for house, questionable for classical

## Anti-patterns
- **Using in classical contexts**: Parallel harmony violates classical voice leading rules
- **Overusing**: Parallel harmony can sound monotonous if used exclusively
- **Ignoring context**: What works in house may not work in jazz or classical

## Key Takeaways
1. Parallel harmony: each note moves the same semitones in the same direction
2. It's essentially a transposed copy of one chord shape
3. Common in house/deep house music
4. Sampled chords, chord memory synths, and chord MIDI tools create parallel harmony
5. Genre-specific — great for electronic, questionable for classical
