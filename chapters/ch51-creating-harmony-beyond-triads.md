# Chapter 51: Creating Harmony 2: Beyond Triads

## Core Idea
Seventh chords (dominant 7th wants to resolve to I). 9th, 11th, 13th chords for color. Alterations (b5 = #11). Quartal/quintal chords (stacked 4ths/5ths). Secundal/chord clusters. Voice leading between dense chords.

## Frameworks Introduced
- **Seventh chords**: Add the 7th scale degree to a triad. Dominant 7th (major triad + minor 7th) wants to resolve to I.
- **Extended chords**: 9th, 11th, 13th chords add color. Each extension adds a new scale degree.
- **Alterations**: b5 = #11, b9, #9, #11, b13 — alterations add tension and color.
- **Quartal chords**: Chords built by stacking 4ths (instead of 3rds). Creates open, ambiguous harmony.
- **Quintal chords**: Chords built by stacking 5ths. Even more open than quartal.
- **Secundal/chord clusters**: Chords built by stacking 2nds. Creates dense, dissonant harmony.
- **Voice leading dense chords**: When moving between dense chords, preserve common tones and minimize motion.

## Key Concepts
- **Extensions add color**: Each extension (9th, 11th, 13th) adds a new harmonic color
- **Alterations create tension**: Altered chords create strong pull toward resolution
- **Quartal/quintal ambiguity**: Stacked 4ths/5ths don't clearly indicate major or minor
- **Cluster density**: Chord clusters are the densest possible harmony — every note is adjacent

## Mental Models
- Use extended chords when triads feel too simple
- Use quartal chords when you want open, ambiguous harmony
- Use alterations when you want strong tension and resolution
- Voice leading is essential for dense chords — preserve common tones

## Anti-patterns
- **Adding extensions randomly**: Extensions should serve the harmonic function
- **Ignoring voice leading**: Dense chords without careful voice leading create muddiness
- **Over-altering**: Too many alterations create harmonic confusion

## Key Takeaways
1. Seventh chords add the 7th scale degree — dominant 7th wants to resolve to I
2. 9th, 11th, 13th chords add harmonic color
3. Alterations (b5, #11, b9, #9) add tension
4. Quartal (stacked 4ths) and quintal (stacked 5ths) chords create open harmony
5. Secundal/chord clusters are the densest harmony — use voice leading to move between them
