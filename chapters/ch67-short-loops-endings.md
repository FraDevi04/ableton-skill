# Chapter 67: Short Loops as Endings

## Core Idea
Isolate a 1-2 bar sub-passage from the closing phrase. Repeat it a set number of times. Optionally shorten further (2 bars → 1 bar). Creates illusion of speeding up toward the end. Add unique events for drive.

## Frameworks Introduced
- **Short loop ending**: Isolate a 1-2 bar sub-passage from the closing phrase. Repeat it a set number of times. This creates a clear, satisfying ending.
- **Loop shortening**: Optionally shorten the loop further with each repeat (2 bars → 1 bar). This creates an illusion of speeding up toward the end.
- **Unique events for drive**: Add unique sounds or events to the loop to create forward momentum and prevent monotony.

## Key Concepts
- **Sub-passage isolation**: Extract a small fragment from the closing phrase — not the whole phrase
- **Repeat count**: The number of repeats determines the ending length
- **Acceleration illusion**: Shortening the loop creates a sense of acceleration
- **Drive events**: Unique events (fills, hits, effects) add energy to the ending

## Mental Models
- Use short loop endings when you need a clear, satisfying ending
- Think of it as "zooming in" on a small fragment and repeating it
- Shortening creates acceleration — use it to build energy toward the end

## Anti-patterns
- **Too long**: A 4-bar loop repeated 10 times is boring
- **Too short**: A 1-bar loop repeated 3 times may feel abrupt
- **No drive events**: Pure repetition without unique events creates monotony

## Key Takeaways
1. Isolate a 1-2 bar sub-passage from the closing phrase
2. Repeat it a set number of times for a clear ending
3. Optionally shorten the loop (2 bars → 1 bar) for acceleration illusion
4. Add unique events (fills, hits, effects) for drive
5. Short loop endings create satisfying, professional-sounding conclusions
